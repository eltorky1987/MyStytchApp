# 🚀 برنامج أجينا للمحاسبة الذكية (Agina Smart v2.2)

نظام محاسبي متكامل لإدارة المبيعات، مصمم بلغة **Java** ومخصص للعمل بامتياز على بيئة **Termux**.

---

## 🌟 المميزات الرئيسية
* **🛡️ وصول آمن:** محمي بواسطة مفتاح وصول خاص لضمان السرية.
* **⚙️ نظام ذكي:** يتعامل مع المدخلات بنظام يمنع الأعطال.
* **📊 تخزين فوري:** يتم حفظ جميع البيانات وتتبعها عبر ملف `database.csv`.

---

## 🛠 كيفية التشغيل
للمبرمجين والمستخدمين، اتبع الأوامر التالية:
1. **تجميع الكود:** `javac MainActivity.java`
2. **بدء التشغيل:** `java MainActivity`
3. **مفتاح الدخول:** `agina@2026`
